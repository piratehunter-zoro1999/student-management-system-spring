# Student Management System (Spring Boot)

This is a backend project where I converted my console-based student system into a REST API using Spring Boot.

## Features
- Add student
- Get all students
- Get student by ID
- Delete student

## Improvements
- Added structured API response (status, message, data)
- Implemented validation for inputs
- Handled errors using global exception handler

## Tech Used
- Spring Boot
- Spring Data JPA
- H2 Database

## How to Run
- Run StudentManagementSystemApplication.java
- Use Postman to test APIs
